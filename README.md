# Como criar um repositorio no Github
É necessario ter instalado o **git** no pc e utilizar tambem **github desktop** para facilitar

##passo a passo
supondo que voce ja tem um projeto criado,faça o seguinte:
-abrir o cmd dentro da pasta e digitar o comando ```git init```
-ir no github desktop, e escolher a opção ``` Add existing repository``` e selecionar a pasta do projeto
-por fim, fazer o commit e publcar. **atencao:** lembrar de deixar sempre publico
